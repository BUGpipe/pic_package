## Virtual Environment: Conda

### Create an env called "cs235" using Python3.9
```
conda create -n cs235 python=3.9
```
![pic](/pic/2.png)

### List all existing env
```
conda env list
```
![pic](/pic/9.png)
### Activate and deactivate
```
conda activate cs235

conda deactivate
```
![pic](/pic/3.png)
### Remove env
```
conda env remove -n cs235
```
